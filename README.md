## Web Scrapy for rutracker.org
Need python3.7 virturalenv and so on

### TL; TD
```
git clone git@github.com:kimp1er/web-scrap-rutracker.git
cd web-scrap-rutracker/
virturalenv .
. ./bin/activate
pip install -r requirements.txt
cd rutracker/
scrapy crawl ruspider 
```
